# Architectural Styles Reference

Detailed style guides for Minecraft building.

## Medieval European

**Characteristics:**
- Half-timbered facades (exposed wood framing)
- Steep gabled roofs (45-60 degrees)
- Small windows with shutters
- Stone foundations, wood upper floors

**Materials:**
- Primary: Oak planks, spruce planks
- Structure: Stripped oak logs, cobblestone
- Roof: Dark oak stairs, spruce stairs
- Details: Oak trapdoors (shutters), lanterns

**Proportions:**
- 2-3 stories typical
- Ground floor slightly larger
- Overhanging upper floors (jetty)
- Roof height = 40-50% of building height

**Example palette:**
```
oak_planks (walls 60%)
stripped_oak_log (framing 25%)
cobblestone (foundation 10%)
dark_oak_stairs (roof)
```

## Gothic

**Characteristics:**
- Pointed arches (not rounded)
- Flying buttresses
- Rose windows
- Tall spires, vertical emphasis
- Ribbed vaults

**Materials:**
- Primary: Stone bricks, deepslate bricks
- Structure: Polished deepslate, smooth stone
- Windows: Black stained glass, gray stained glass
- Details: Stone brick walls, chains

**Proportions:**
- Height:Width ratio 2:1 or greater
- Narrow lancet windows (1:3 ratio)
- Buttresses every 4-6 blocks
- Spire height = 50%+ of tower height

**Example palette:**
```
stone_bricks (walls 60%)
deepslate_bricks (accents 25%)
black_stained_glass_pane (windows 10%)
stone_brick_stairs (roof/details)
```

## Japanese Traditional

**Characteristics:**
- Low-pitched hip roofs with curved eaves
- Sliding doors (shoji screens)
- Raised floors (engawa/veranda)
- Natural materials, minimal decoration
- Asymmetric gardens

**Materials:**
- Primary: Spruce planks, birch planks
- Structure: Dark oak logs, stripped bamboo
- Roof: Dark prismarine stairs, deepslate tile stairs
- Details: Paper (white wool), bamboo

**Proportions:**
- Low buildings (1-2 stories)
- Wide eaves (2-3 block overhang)
- Roof pitch 20-30 degrees
- Tatami rooms: 3×6, 4.5×9, 6×9 blocks

**Example palette:**
```
spruce_planks (walls/floor 60%)
dark_oak_log (structure 20%)
deepslate_tile_stairs (roof 15%)
white_wool (screens 5%)
```

## Modern/Contemporary

**Characteristics:**
- Flat roofs or very low pitch
- Large glass walls
- Clean geometric forms
- Cantilevers and overhangs
- Open floor plans

**Materials:**
- Primary: White/gray concrete, quartz
- Structure: Iron blocks, polished blackstone
- Windows: Glass panes, tinted glass
- Details: Sea lanterns (lighting), iron bars

**Proportions:**
- 2-4 stories
- Floor-to-ceiling windows (2-3 blocks)
- Cantilevered sections extend 2-4 blocks
- Minimal roof detail

**Example palette:**
```
white_concrete (walls 50%)
light_gray_concrete (accents 30%)
glass_pane (windows 15%)
iron_block (structure 5%)
```

## Castle/Fortress

**Characteristics:**
- Thick walls (2-3 blocks)
- Corner towers
- Battlements/crenellations
- Arrow slits
- Gatehouse with portcullis

**Materials:**
- Primary: Stone bricks, cobblestone
- Structure: Deepslate bricks, polished andesite
- Details: Iron bars, chains, anvils
- Roof: Stone brick stairs, deepslate tile stairs

**Proportions:**
- Walls 4-6 blocks thick
- Towers 8-12 blocks diameter
- Crenellations: 2 up, 1 gap
- Arrow slits: 1 wide, 3 tall

**Example palette:**
```
stone_bricks (walls 55%)
cobblestone (base/rough areas 25%)
deepslate_bricks (towers 15%)
iron_bars (windows/portcullis 5%)
```

## Fantasy/Wizard Tower

**Characteristics:**
- Cylindrical towers
- Conical roofs
- Magical elements (glowing blocks)
- Spiral staircases
- Asymmetric additions

**Materials:**
- Primary: Prismarine, purple terracotta
- Structure: Deepslate, end stone
- Windows: Purple/magenta stained glass
- Details: End rods, amethyst, crying obsidian

**Proportions:**
- Tower diameter 7-11 blocks
- Height:diameter ratio 3:1 or greater
- Roof cone height = tower diameter
- Multiple levels with windows

**Example palette:**
```
deepslate_bricks (tower 50%)
prismarine_bricks (accents 25%)
purple_stained_glass (windows 10%)
end_rod (lighting 5%)
amethyst_block (crystals 10%)
```

## Desert/Sandstone

**Characteristics:**
- Domed roofs
- Inner courtyards
- Thick walls for insulation
- Small windows
- Flat rooftops with parapets

**Materials:**
- Primary: Sandstone, smooth sandstone
- Structure: Cut sandstone, terracotta
- Roof: Sandstone stairs, orange terracotta
- Details: Terracotta patterns, blue accents

**Proportions:**
- Low buildings (1-2 stories)
- Dome height = dome radius
- Courtyard 40-60% of footprint
- Wall thickness 2 blocks

**Example palette:**
```
smooth_sandstone (walls 60%)
cut_sandstone (trim 20%)
orange_terracotta (roof/dome 15%)
cyan_terracotta (accents 5%)
```
