# Block Palette & Color Theory Guide

Master color combinations and material palettes for professional Minecraft builds.

---

## Color Theory Basics

### The Color Wheel in Minecraft

Minecraft blocks roughly map to color theory:

```
           WHITE
      Birch   Quartz   Diorite
           ↑
WARM ←     |     → COOL
Red        |       Blue
Orange     |       Cyan
Yellow     |       Purple
           ↓
          BLACK
    Deepslate   Obsidian
```

### Color Relationships

**Complementary** (opposite on wheel) - High contrast, vibrant
- Red + Cyan: `red_terracotta` + `prismarine`
- Orange + Blue: `orange_concrete` + `blue_concrete`
- Yellow + Purple: `yellow_terracotta` + `purple_concrete`

**Analogous** (adjacent on wheel) - Harmonious, calm
- Warm: `orange_terracotta` + `red_terracotta` + `brown_terracotta`
- Cool: `blue_concrete` + `cyan_terracotta` + `light_blue_concrete`
- Earth: `brown_concrete` + `orange_terracotta` + `yellow_terracotta`

**Triadic** (equidistant on wheel) - Balanced, dynamic
- Primary: Red + Yellow + Blue terracotta
- Secondary: Orange + Green + Purple concrete

**Monochromatic** (shades of one color) - Elegant, unified
- Grays: `white_concrete` → `light_gray` → `gray` → `black_concrete`
- Browns: `birch_planks` → `oak_planks` → `dark_oak_planks`

## The 60-30-10 Rule

For any build, use:
- **60% Primary** - Main walls, large surfaces
- **30% Secondary** - Structural elements, trim, roofs
- **10% Accent** - Details, decorations, highlights

### Example Application

**Medieval Cottage:**
- 60% `oak_planks` (walls)
- 30% `cobblestone` (foundation, chimney) + `dark_oak_log` (timber frame)
- 10% `lantern` + `glass_pane` + `flower_pot` (details)

**Modern House:**
- 60% `white_concrete` (walls)
- 30% `gray_concrete` (base, frames) + `glass` (windows)
- 10% `birch_planks` (door, deck) + `sea_lantern` (lights)

## Material Categories

### Natural Stone Spectrum

**Light to Dark:**
```
Lightest → → → → → → → → Darkest

diorite → calcite → smooth_stone → stone → andesite → cobblestone → deepslate → blackstone
```

**Warmth Scale:**
```
Warm                                  Cool
←─────────────────────────────────────→
granite  sandstone  stone  andesite  prismarine  deepslate
```

### Wood Spectrum

**Light to Dark:**
```
birch → bamboo → oak → jungle → acacia → spruce → dark_oak → crimson → mangrove
```

| Wood | Tone | Best For |
|------|------|----------|
| Birch | Light, yellow | Modern, Japanese, beach |
| Bamboo | Light, green | Japanese, tropical |
| Oak | Medium, warm | Universal, medieval |
| Spruce | Medium, orange | Mountain, medieval |
| Dark Oak | Dark, brown | Gothic, medieval |
| Crimson | Dark, red | Nether, fantasy |
| Warped | Dark, teal | Fantasy, underwater |

### Concrete Colors

**Sorted by warmth and saturation:**

| Category | Colors | Use For |
|----------|--------|---------|
| Neutrals | white, light_gray, gray, black | Modern, industrial |
| Warm Brights | red, orange, yellow | Accents, playful |
| Cool Brights | lime, cyan, light_blue, blue | Modern, pool areas |
| Muted | brown, green, purple, magenta, pink | Gradients, details |

### Terracotta Colors

Terracotta has more muted, earthy tones than concrete. Better for:
- Desert builds
- Pueblo/Adobe style
- Natural gradients
- Ancient/weathered looks

**Best terracotta combinations:**
- Desert: `orange` + `yellow` + `brown` + `white`
- Mediterranean: `white` + `cyan` + `light_blue` + `orange`
- Autumn: `red` + `orange` + `brown` + `yellow`

## Gradient Techniques

### Vertical Gradients

Build from dark to light going up (natural lighting):

**Stone Wall:**
```
Top:    smooth_stone, polished_andesite
Middle: stone_bricks
Bottom: cobblestone, mossy_cobblestone
```

**Desert Tower:**
```
Top:    smooth_sandstone
Middle: sandstone
Bottom: cut_sandstone, chiseled_sandstone
```

### Horizontal Gradients

Blend between materials across a surface:

**Weathered Wall:**
```
Random mix: 60% stone_bricks + 25% cracked_stone_bricks + 15% mossy_stone_bricks
```
Use WorldEdit pattern: `60%stone_bricks,25%cracked_stone_bricks,15%mossy_stone_bricks`

**Copper Aging (top to bottom):**
```
copper_block → exposed_copper → weathered_copper → oxidized_copper
```

### Noise/Random Textures

For organic, weathered looks:

**Stone floor:**
```
50%stone_bricks,20%cobblestone,15%andesite,10%gravel,5%mossy_cobblestone
```

**Dirt path:**
```
40%coarse_dirt,30%dirt,20%podzol,10%gravel
```

## Contrast Principles

### Value Contrast

Use light/dark differences for visibility:

**High Contrast (readable):**
- `white_concrete` walls + `dark_oak_log` trim
- `quartz_block` floor + `black_concrete` border

**Low Contrast (subtle):**
- `stone_bricks` walls + `andesite` trim
- `oak_planks` floor + `spruce_planks` border

### Texture Contrast

Mix smooth and rough textures:

**Smooth vs Rough:**
- `smooth_stone` pillars + `cobblestone` walls
- `polished_deepslate` floor + `deepslate_bricks` border
- `quartz_block` base + `quartz_bricks` pattern

### Temperature Contrast

Warm and cool materials together:

**Warm accent in cool build:**
- `prismarine` temple + `copper` details
- `deepslate` castle + `shroomlight` lights

**Cool accent in warm build:**
- `sandstone` palace + `cyan_terracotta` tiles
- `spruce` lodge + `light_blue_stained_glass`

## Common Mistakes

### 1. Too Many Materials
**Problem:** Using 6+ different block types
**Solution:** Limit to 3-4 main materials per area

### 2. No Accent Color
**Problem:** Build looks flat and boring
**Solution:** Add 10% accent for visual interest

### 3. Wrong Proportions
**Problem:** Trim overpowers walls, or details invisible
**Solution:** Follow 60-30-10 rule

### 4. Ignoring Texture
**Problem:** All smooth or all rough surfaces
**Solution:** Mix textures deliberately

### 5. Mismatched Temperature
**Problem:** Random warm/cool blocks clashing
**Solution:** Choose warm OR cool base, accent with opposite

### 6. No Gradation
**Problem:** Sharp transitions between colors
**Solution:** Add intermediate tones

## Pattern Syntax for WorldEdit

### Random Mix
```
50%stone_bricks,30%cobblestone,20%andesite
```

### Layered
```
stone_bricks^cobblestone     (cobblestone on stone_bricks)
```

### Block States
```
oak_stairs[facing=north,half=bottom]
```

### Combined
```
50%stone_bricks,50%stone_bricks[waterlogged=true]
```
