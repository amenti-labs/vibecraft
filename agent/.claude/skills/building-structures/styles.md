# Architectural Styles Reference

## Medieval European
Half-timbered facades, steep gabled roofs (45-60°), small windows with shutters, stone foundations.

**Materials**: oak_planks (60%), stripped_oak_log (25%), cobblestone (10%), dark_oak_stairs (roof)
**Proportions**: 2-3 stories, overhanging upper floors (jetty), roof 40-50% of building height

## Gothic
Pointed arches, flying buttresses, rose windows, tall spires, vertical emphasis, ribbed vaults.

**Materials**: stone_bricks (60%), deepslate_bricks (25%), black_stained_glass_pane (10%), stone_brick_stairs
**Proportions**: Height:Width 2:1+, lancet windows 1:3, buttresses every 4-6 blocks, spire 50%+ of tower

## Japanese Traditional
Low-pitched hip roofs with curved eaves, sliding doors (shoji), raised floors, natural materials.

**Materials**: spruce_planks (60%), dark_oak_log (20%), deepslate_tile_stairs (15%), white_wool (5%)
**Proportions**: 1-2 stories, 2-3 block overhang, 20-30° pitch, tatami rooms: 3×6, 4.5×9, 6×9

## Modern/Contemporary
Flat/low-pitch roofs, large glass walls, clean geometric forms, cantilevers, open floor plans.

**Materials**: white_concrete (50%), light_gray_concrete (30%), glass_pane (15%), iron_block (5%)
**Proportions**: 2-4 stories, floor-to-ceiling windows 2-3 blocks, cantilevers 2-4 blocks

## Castle/Fortress
Thick walls (2-3 blocks), corner towers, battlements, arrow slits, gatehouse with portcullis.

**Materials**: stone_bricks (55%), cobblestone (25%), deepslate_bricks (15%), iron_bars (5%)
**Proportions**: Walls 4-6 blocks thick, towers 8-12 diameter, crenellations: 2 up/1 gap, arrow slits: 1×3

## Fantasy/Wizard Tower
Cylindrical towers, conical roofs, magical elements (glowing blocks), spiral staircases.

**Materials**: deepslate_bricks (50%), prismarine_bricks (25%), purple_stained_glass (10%), end_rod (5%), amethyst_block (10%)
**Proportions**: Tower diameter 7-11, height:diameter 3:1+, roof cone = tower diameter

## Desert/Sandstone
Domed roofs, inner courtyards, thick walls for insulation, small windows, flat rooftops with parapets.

**Materials**: smooth_sandstone (60%), cut_sandstone (20%), orange_terracotta (15%), cyan_terracotta (5%)
**Proportions**: 1-2 stories, dome height = dome radius, courtyard 40-60% of footprint, walls 2 blocks thick
